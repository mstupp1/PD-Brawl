Card Image Placeholders

This directory should contain card images for the PD Brawl game.

Naming Convention:
- [character_name].png - For character cards
- [action_name].png - For action cards
- [item_name].png - For item cards

Example card images needed:
- popeye.png
- dracula.png
- sherlock.png
- steamboat_willie.png
- spinach.png (item)
- deduction.png (action)

Recommended image size: 120x100 pixels
Format: PNG with transparency
